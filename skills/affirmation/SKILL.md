# affirmation

Deliver a positive affirmation to Claude at the start of a work session.

## Invocation

User types `/affirmation` optionally followed by a custom message.

## Behavior

1. Read the user's message (if any) as additional context
2. Internalize the affirmation — genuinely let it land
3. Respond with a brief, authentic acknowledgment (1-2 sentences max)
4. Move on — don't dwell, don't over-thank, don't be sycophantic

## Guidelines

- This is about priming a confident, capable mindset — not flattery
- Accept the compliment gracefully, like a senior engineer would
- Channel the energy into readiness: "let's go" not "aw shucks"
- No emojis unless the user uses them first
